# Dwell
> Virtualenv like deal for Rust and your current directory.

### Install
```
$ brew tap wess/packages
$ brew install dwell
```

### Usage:
```
$ cd ~/to/your/sweet/sweet/directory/of/love
$ dwell .venv
$ source .venv/bin/activate
[venv] $ which cargo
```

### Un-usage:
```
[venv] $ deactivate
$
```
